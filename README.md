🇮🇩 Clear Drop Items (Sampah Masyarakat!)
Efficient Lag-Clearing Skript for Your Minecraft Server
📖 Overview
Clear Drop Items (Sampah Masyarakat!) is a lightweight and high-performance Skript designed to keep your server running at peak performance. By automatically cleaning up dropped items that clutter the ground, this script significantly reduces entity lag—perfect for Survival, SMP, or Factions servers.



Fun Fact about the name: In Indonesia, "Sampah Masyarakat" literally translates to "Scum of Society." It's a local inside joke because dropped items that cause lag are the "scum" of the server world!

🚀 Key Features
Automatic Cleanup: Runs a global cleanup every 20 minutes (customizable).
Gradual Warning System: Sends broadcast alerts at 1 minute, 30 seconds, and 10 seconds before clearing, so players can save their valuables.
Manual Control: Admins can trigger a cleanup instantly via command.
Live Reports: Shows the exact number of entities removed in the chat.
Audio Feedback: Plays an XP-Orb sound effect once the cleanup is successful.
Fully Configurable: Easily change the prefix ([&c&lSAMPAH]) and permissions in the options.
💻 Commands & Permissions
Command	Aliases	Description	Permission
/sampah	/clearlag, /cl	Manually clears all dropped items	admin.sampah
⚙️ Easy Configuration
You can customize the script directly at the top of the file:



options:

    p: &8[&c&lSAMPAH&8]       # Chat Prefix

    perm: admin.sampah        # Required Permission



🛠️ Installation Guide
Ensure you have the Skript Plugin installed.
Download sampah.sk.
Upload the file to your server: /plugins/Skript/scripts/.
Run the command: /sk reload sampah.
You are all set!
